# 新闻分类 Demo - PRD

## 1. 核心目标
- 提供一个可本地运行的 Streamlit demo，用于展示新闻分类能力。
- 支持粘贴 WP 后台“文字”模式原始内容（含 HTML、caption、图片链接）并完成清洗。
- 通过 LLM 对标题与正文分块进行分类分析，输出最终分类结论。

## 2. 功能范围
- 输入：标题 + WP 原始正文（手动粘贴）。
- 处理：
  - 内容清洗（去 caption、去图片标签、HTML 实体解码、提取段落/副标题）。
  - 分割策略（标题单独、按 h2/h3/h4 分块、无副标题则每 3 段分块）。
  - LLM 分块分类（标题、chunk）。
  - 总决策聚合（低置信度降权，不直接丢弃）。
- 输出：
  - 清洗后的文本预览。
  - 每个 chunk 的分类结果。
  - 最终分类 JSON 与可读摘要。

## 3. 非目标（首版不做）
- 不接入真实 WP API 的导入导出。
- 不做批量任务调度与异步队列。
- 不做生产级鉴权与权限系统。

## 4. 必须完成的指标
- 可处理典型 WP 粘贴输入并得到可读段落。
- chunk 分割策略按约定执行且可视化。
- LLM 输出结构化 JSON（含容错重试）。
- 最终结果使用降权策略处理低置信度 chunk。

## 5. 业务分类体系
- 主分类：财经、体育、娱乐、时事、科技、其他。
- 支持次分类（数组）。
